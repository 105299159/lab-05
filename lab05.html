Headings and Paragraphs

Headings and paragraphs are BLOCK elements.

Headings

There are 6 levels of heading. Headings are logical markup. Do not make levels. If you want to emphasize words or phrases, don't change the font - change the tag.

Paragraphs

Paragraphs contain the body text of a web page. You cannot put another block element inside a paragraph.

Special Characters

While HTML supports many named special characters like < (<), XML only supports 5 normally: < > & ' and ". So to keep you HTML compliant you need to use numbers for all the others.

Lists

There are three types of lists: Unordered and Ordered lists, and Definition lists.

Unordered List

  • First bullet point
  • Second bullet point
  • Third bullet point

Ordered List

  1. First item
  2. Second item
  3. Third item

Definition List

First term (or name)
Basic description (or definition, or value)
Second term (or name)
Longer description (or definition, or value)

Lists can also be nested inside each other.

  1. First item
    1. First sub-item
    2. Second sub-item
    3. Third sub-item
  2. Second item
  3. Third item
